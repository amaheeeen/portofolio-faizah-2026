## 🌟 About This Project

This repository houses the personal portfolio for **Nur Faizah Syam**, a Brand Strategy Leader with over 9 years of experience in Social Media Ads and Market Expansion. Rather than a standard flat design, this portfolio pushes the boundaries of UI/UX by implementing a rich, tactile **Skeuomorphic (Sage & Sand Edition)** design system using custom CSS.

### ✨ Key Features

- **Skeuomorphic Design System**: Custom CSS variables orchestrating deep, multi-layered `box-shadows` to create "extruded" (timbul) and "pressed" (tenggelam) physical effects.
- **Dynamic Theming**: Fluid transitions between beautiful, highly contrasted light and dark modes holding true to the physical material look.
- **Interactive Micro-Animations**: Buttons and inputs that physically depress when clicked, giving a satisfying, tactile user experience.
- **Zero Framework Bloat**: Logic is handled exclusively via Vanilla JavaScript, routing away from heavy JS frameworks for absolute maximum performance.
- **Responsive & Accessible**: Fully fluid layout ensuring perfect proportions from ultra-wide 4K monitors down to mobile devices.

---

## 🎨 The "Sage & Sand" Aesthetic

Inspired by organic materials and high-end physical objects:

- **Backgrounds**: Warm Beige (`#F2E8C6`) with soft paper-like textures.
- **Hero/Accent**: Dusty Teal/Sage (`#5F8670`) for action buttons and visual anchors.
- **Shadows**: Crafted perfectly at a 135-degree angle simulating a top-left light source.
- **Highlights**: Crisp white inner bevels mimicking raised geometry.

## 💻 Tech Stack Highlights

- **Markup**: Semantic `HTML5`
- **Styling**: `Tailwind CSS` (CDN) integrated seamlessly with heavy custom Vanilla `CSS3` for the 3D skeuomorphism elements.
- **Interactivity**: `Vanilla JS` serving Lightbox modals, smooth scrolling, and systemic theme toggling.
- **Icons**: [Devicon](https://devicon.dev/) & [Icons8](https://icons8.com)

## 🛠️ Usage & Deployment

Being entirely static, deployment is as simple as hosting the directory on any web server:

```bash
# Clone the repository
git clone https://github.com/nfzh_/portofolio-faizah-2026.git

# Navigate into the project
cd portofolio-faizah-2026

# Serve locally using Node/Python/LiveServer
npx serve .
# Or simply open index.html in your browser!
```
